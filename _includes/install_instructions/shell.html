The Bash Shell

Bash is a commonly-used shell that gives you the power to do tasks more quickly.

  • Windows
  • MacOS
  • Linux
  1. We recommend to use Windows Subsystem for Linux (WSL) to use Bash in Windows. Follow the WSL installation instruction to run a Linux distribution in Windows.
  2. To enable WSL, you'll need :
    1. Windows 10 version 2004 and higher (Build 19041 and higher) or Windows 11.
    2. Admin access to your OS
    3. Hardware which support Hyper-V
  3. We recommend installing Ubuntu-20.04 as your distribution

The default shell in some versions of macOS is Bash, and Bash is available in all versions, so no need to install anything. You access Bash from the Terminal (found in /Applications/Utilities). See the Git installation video tutorial for an example on how to open the Terminal. You may want to keep Terminal in your dock for this workshop.

To see if your default shell is Bash type echo $SHELL in Terminal and press the Return key. If the message printed does not end with '/bash' then your default is something else and you can run Bash by typing bash

If you want to change your default shell, see this Apple Support article and follow the instructions on "How to change your default shell".
